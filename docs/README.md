### Permission Schemes

[Документация Permission Schemes API](./permissionscheme/)

Документация по REST API для работы со схемами разрешений в Jira. Содержит описание всех endpoints для создания, получения, обновления и удаления схем разрешений и их разрешений. Используется для управления правами доступа к проектам и их функциям.

### Project Permission Schemes

[Документация Project Permission Schemes API](./projectsecurityschemes/)

Документация по REST API для работы со схемами разрешений на уровне проектов. Содержит endpoints для получения и назначения схем разрешений проектам, получения схем безопасности issue и уровней безопасности. Используется для управления правами доступа на уровне конкретных проектов.
